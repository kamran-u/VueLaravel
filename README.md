## Open weather api - https://openweathermap.org/guide
Create account (free) and supply api key to WEATHER_API_KEY in .env file

## Pre-requisite
- database: weather
- composer install
- php artisan migrate
- run cities sql (databse > seeds > cities.sql) on database: weather  


## Running the app
php 7.2 is required
